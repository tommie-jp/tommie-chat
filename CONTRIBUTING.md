# Contributing to tomChat

tomChat への貢献に興味を持っていただきありがとうございます！

## 貢献の方法

### バグ報告

- [Issues](../../issues) からバグ報告を作成してください
- 再現手順、期待する動作、実際の動作を記載してください
- 可能であればスクリーンショットやブラウザのコンソールログを添付してください

### 機能リクエスト

- [Issues](../../issues) から機能リクエストを作成してください
- どのような機能が欲しいか、なぜ必要かを説明してください

### プルリクエスト

1. このリポジトリをフォークしてください
2. フィーチャーブランチを作成してください（`git checkout -b feature/your-feature`）
3. 変更をコミットしてください（`git commit -m "Add your feature"`）
4. ブランチをプッシュしてください（`git push origin feature/your-feature`）
5. プルリクエストを作成してください

### PR のガイドライン

- 1つのPRにつき1つの変更に絞ってください
- 既存のコードスタイルに従ってください
- テストが通ることを確認してください（`npm test`）

## 開発環境のセットアップ

```bash
# リポジトリをクローン
git clone https://github.com/open-tommie/tom-chat.git
cd tom-chat

# 依存関係のインストール
npm install

# 開発サーバーの起動
npx vite
```
