# リバーシ UART プロトコル仕様（移管済み）

## 移管理由

UART プロトコル仕様は tommieChat 専用ではなく、自作 CPU（FPGA / MCU /
ディスクリート CPU 等）の製作者が独立して参照・実装できる仕様書として、
別リポジトリで管理する方針に変更しました。
