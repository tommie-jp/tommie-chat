# 検討事項: セリフ機能

他のモバイルアプリ/MMOを参考にした改善候補。

## 改善候補一覧

| 項目 | 他アプリの例 | tommieChat の現状 | 優先度 |
| --- | --- | --- | --- |
| セリフ自動消去 | FF14: 約15秒で消える、PSO2: 同様 | 消えない(手動クリアのみ) | 高 |
| 文字数制限の表示 | LINE: 残り文字数表示、Discord: 2000/4000文字 | 制限なし・表示なし | 中 |
| 入力中表示 | Discord: 「〇〇が入力中...」 | なし | 低 |
| 送信時の音/振動 | LINE: 送信音、多くのMMO: SE | なし | 低 |
| セリフ履歴の上矢印復元 | Discord/Slack: 上キーで直前メッセージ編集 | なし | 低 |
| テキスト選択コピー | ほぼ全アプリ: 吹き出しタップでコピー | なし | 低 |

## セリフ自動消去の参考値

| ゲーム/アプリ | 消去までの時間 |
| --- | --- |
| Final Fantasy XIV | 約15秒 |
| PSO2 | 約10秒 |
| Maple Story | 約10秒 |
| アメーバピグ | 約30秒 |

一般的には10〜30秒で消去。新しいセリフを送信したらタイマーリセット。
