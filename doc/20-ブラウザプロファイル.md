# ブラウザサイド プロファイル手順

## 1. 組み込みプロファイラ（コンソール関数）

ブラウザの DevTools コンソールで以下の関数を使用する。

### 基本操作

```js
profileStart()   // 計測開始（DevTools Timings にも mark/measure 出力）
profileStop()    // 計測停止
profileDump()    // 収集データのサマリを表示
```

### 出力内容

`profileDump()` は 3 種類のデータを表示する。

**1) レンダーループ（console.table）**

| name | 説明 |
|------|------|
| playerMove | 自キャラ移動処理 |
| remoteAvatars | 他プレイヤーアバター補間 |
| npc | NPC 処理 |
| total | 上記合計 |
| avatarCount | アバター数 |

各項目に avg / p95 / max (ms) が表示される。

**2) onmatchdata**

```
onmatchdata: 68 calls/s, total=0.7ms/s, max=0.20ms
```

WebSocket メッセージ（位置同期等）の処理コスト。

**3) userList**

```
userList: 1 renders/s, total=3.0ms/s, max=3.00ms, users=236
```

プレイヤーリスト DOM 再構築のコスト。

### リアルタイム表示

デバッグオーバーレイの `Profile` 行に常時表示される（`profileStart()` 不要）。

```
pl=0.00 rm=0.05 npc=0.00 tot=0.06ms | msg=68/s 0.7ms | ul=1/s 3ms n=236
```

## 2. 負荷テストとの組み合わせ

```bash
# ブラウザでログイン後、コンソールで:
profileStart()

# 別ターミナルでテスト実行
./doTest-sustain.sh -n 100 -t 30

# テスト終了後、コンソールで:
profileStop()
profileDump()
```

## 3. Chrome DevTools Performance タブ

手動で詳細な計測を行う場合:

1. DevTools → パフォーマンス タブ
2. 記録ボタン (●) を押す
3. 数秒〜十数秒操作する
4. 停止ボタンを押す
5. **Bottom-Up** タブで関数ごとの所要時間を確認
6. **Call Tree** タブで呼び出し階層を確認
7. **Timings** レーンに `profileStart()` で出力した mark/measure が表示される

### よく確認するポイント

- **Main** スレッドの長いタスク（赤い三角マーク）
- **GPU** レーンの描画負荷
- **Frames** レーンのフレームドロップ（赤枠）
