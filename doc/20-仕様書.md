# tommieChat仕様書

## オセロ

- クライアント:「7b.オセロ」パネル
  - ゲームロビー
    - テーブルカラム
    - No
      - 整数３桁を表示する。例：001
    - 日時
    - オーナー
      - ゲームのオーナー（作成者）の表示名を表示する。
      - 画面幅が狭いときは表示しない。
    - コメント
      - 左にスクロール
      - 表示する文字列：時分 オーナ表示名 セリフ
        - オーナ表示名は太い文字でオーバレイチャットと同じ色で表示する。
        - セリフ初期値
          - 'がゲーム相手を募集中です！'
      - 閲覧ボタン
        - コメント欄と閲覧ボタンは密着させる。
        - 参加ボタンの左に表示。
      - 参加ボタン
        - 参加ボタンは表の右端に密着させる。
    - ゲーム（ゲーム中、応募中）がないとき
      - オーナー欄を表示しない
      - No欄, 日時欄は空白にする。
      - コメント欄に`*** 新ゲームを開始しよう！ ***`を赤色、太文字で表示する
      - コメント欄を左自動スクロールする
    - BUG
      - オーナーカラムラベル、コメントカラムラベルが重ならないようにする
  - 履歴
    - 黒欄
    - 白欄
    - 黒欄、白欄共通
      - 表示名はオーバレイチャットの表示名と同じ色で表示する。
    - スコア欄
      - 半角6文字分の幅、センタリング
    - 決着欄
      - センタリング
    - 表全体
      - スマホ：ドラッグで縦スクロールに対応
        - カラムラベルはスクロールしない
          - ドラッグしたときにカラムラベルは縦に移動しない。
        - 「履歴」ラベル下部と「No」カラムラベル上部の間に表の値が見切れている。
          - 見きれないようにする。（非表示にする）
          - 表の値のスクロール範囲はカラムラベル下部までにする。
            - カラムラベルと表の値が重ならないようにする。
      - スマホ：ドラッグで横スクロールに対応
        - カラムラベル（No以外）もスクロールする
        - No欄の値はスクロールしない
          - ドラッグしたときに横に移動しない。
  - 日時の更新アルゴリズム
    - １秒ごとに日時を更新する。
      - 時間間隔にはtickを用いる。setIntervalは使わない。
      - 日時の変化があるときだけ、表示項目を更新する。
        - 変化がないときは更新しない（チラツキ防止）
  - No
    - サーバDBが管理するオセロゲーム番号 % 1000 を表示する。3桁
    - ツールチップにはゲーム番号そのママ表示する。
  - ゲームロビー、履歴の表の幅はパネル幅にする
  - パネルリサイズでサーバへRPC、websocket通知が発生しないこと。

  - 「対戦相手を待っています」パネル
    - オセロゲーム番号を画面左上に表示する
      - 例："ゲーム番号:012"
