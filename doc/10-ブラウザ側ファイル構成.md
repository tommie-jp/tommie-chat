## 依存関係

```text
main.ts → GameScene
GameScene → AvatarSystem, CloudSystem, NPCSystem, AOIManager
GameScene → UIPanel(setupHtmlUI), DebugOverlay(setupDebugOverlay)
GameScene → NakamaService, ChunkDB, WorldConstants

UIPanel      → GameScene (import type), WorldConstants
DebugOverlay → GameScene (import type), WorldConstants
NPCSystem    → AvatarSystem
AOIManager   → NakamaService, WorldConstants
```

`UIPanel` と `DebugOverlay` は `import type { GameScene }` で型のみ参照し、循環依存を回避しています。
